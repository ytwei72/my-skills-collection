# edu-typeset-skill

按出版印刷标准将文档排版为专业 `.docx`，并在文末生成「附录：知识点/规则总结」。

> **说明**：本文件供**人类**查阅触发方式与示例指令。Agent 加载本 skill 时**不会**自动注入本文；执行排版任务时亦不应读取本文件（见 `SKILL.md`）。

---

## 如何触发

Cursor 根据 `SKILL.md`  frontmatter 中的 **`description`** 做意图匹配。当你的请求体现出「出版级 docx 排版、格式规范化、教辅/讲义专业排版」时，Agent 会读取并执行 `SKILL.md`。

- **不必**写 skill 路径或 `@edu-typeset`（除非团队另有约定）
- **建议**写清任务意图，并附上**源文件路径**

### 容易触发的关键词

| 类型 | 示例说法 |
|------|----------|
| 明确排版 | 出版排版、专业格式、印刷标准、重新排版、格式规范化 |
| 隐含也会触发 | 整理格式、统一版式、做成可印刷版、教辅排版、讲义排版 |
| 带材料类型 | 英语教辅、练习册、教案、讲义、报告 |

### 通常不会走本 skill 的情况

- 只要 **Markdown/HTML 网页** → 使用 `md-to-rich-html` 等
- **专利交底书、五书形式诊断** → 使用对应 patent skill
- **写入飞书文档** → 使用 `feishu-doc-writer`
- 仅改少量文字或单处边距、无「整份出版级排版」意图

## 用户指令示例

### 1. 英语教辅（倾向使用 `assets/template-en-edu.js`）

```
请用出版排版标准处理这份英语辅导材料：
Miscellaneous\教材出版\Tom.docx
一定要先做理解，再重新排版输出。
```

> **效果说明**：「先做理解」会让 Agent 在排版前充分分析文档的章节结构、内容模块与语法主题，从而做出更精准的层级拆分、模块归类与附录提炼，而不是机械套用模板。**推荐在文档结构较复杂时使用此句式。**

```
请用出版排版标准处理这份英语辅导材料，输出 docx：
Miscellaneous\教材出版\Tom.docx
要求：章节标题清晰、练习题/答案/解析分块，文末加「附录：知识点总结」。
```

```
把 Aladdin 这本练习册按印刷标准重新排版，和仓库里「Aladdin 排版说明」一致，生成 _排版版.docx。
```

### 2. 未说「排版」但意图明确

```
这份教案格式很乱，请统一成可打印的正式版 docx，并附规则总结附录。
路径：Miscellaneous\教材出版\xxx.docx
```

```
这是讲义初稿，请按教辅出版规范整理版式（字体、页眉页脚、表格），输出 docx。
```

### 3. 通用文档（报告 / 教案 / 论文式）

```
对下面这份报告做专业排版，A4、页眉页脚、标题层级，最后加「关键术语与结论速查」附录：
path\to\report.docx
```

```
把这份 .md 讲义排成出版级 .docx（可先转内容再排版），输出到同目录。
```

### 4. 指定输出与约束

```
请按 edu-typeset 流程排版「野性的呼唤」相关材料，保留中英对照结构，答案用绿色框、提示用米黄框，验证通过后再给我文件路径。
```

```
只排版正文，附录里做「易错点对比表」，不要添加原文没有的知识点。
```

### 5. 结合仓库内排版说明

```
按 Miscellaneous\教材出版\Aladdin 排版说明.md 的标准，对 Aladdin - 4页.docx 做出版排版并生成排版版。
```

## 写好指令的小技巧

1. **给出源文件路径**（`.docx` / `.md` / `.txt` 等）
2. **说明材料类型**（英语教辅 / 教案 / 报告），便于选择模板
3. **说明交付物**：要 `.docx`、是否使用 `_排版版` 后缀、是否需要附录
4. **可选约束**：是否对照某份「排版说明」、色块含义、附录不得超出原文
5. **结构复杂时加上**：「一定要先做理解，再重新排版输出」——让 Agent 在动手前先解析文档层次与内容模块，排版质量更高

## 自动与显式调用

本 skill 为项目**默认可自动发现**的 skill之一（`disable-model-invocation: false`），涉及排版、格式规范化时会自动加载。

也可显式调用：

- `/edu-typeset`（以 Cursor 实际注册的 skill 名为准）
- 或明确说：「按 `.cursor/skills/edu-typeset-skill/SKILL.md` 执行出版排版」
